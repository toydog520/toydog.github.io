# 合約架構

## 一、基本資料（甲方 & 寵物）

**寵物資料**
- 寵物名、品種、晶片號碼
- 性別：公 / 母
- 年齡、體重
- 絕育情形：已絕育 / 未絕育
- 目視外觀：正常 / 異常
- 定期疫苗接種：是 / 否

**疾病史選項**
心臟病、癲癇、氣喘、氣管塌陷、白內障、心絲蟲、手術傷口、關節問題、血便、血尿、懷孕、傳染性疾病、骨折、腹積水、腹膜炎、腦炎、外傷、無、其他

**寵物個性選項**
親人、親狗、緊張、護食、護玩具、有攻擊性、咬狗、咬貓、吠叫、無攻擊性

**飼主資料**
- 飼主姓名、聯絡電話
- 緊急聯絡人、聯絡電話
- 甲方是否為飼主本人
- 指定獸醫診療機構

---

## 二、服務內容與費用

- 服務日期：起訖時間
- 服務項目：安親 / 住宿 / 洗澡 / 剪毛 / 其他
- 費用：金額、給付方式（現金 / 轉帳 / 已付清 / 分期）
- 後續各次服務由附件「客戶資料卡」逐次記載

---

## 三、雙方權利與義務

**甲方義務**
- 如實告知寵物健康狀況（患病、疫苗、飲食、個性、懷孕、攻擊性）
- 幼齡/高齡寵物需個別評估，不適合者乙方可婉拒
- 提前準備足夠糧食、藥物，並告知使用方式
- 入館前完成基礎清潔（洗澡、指甲修剪）
- 依約定時間接回寵物

**乙方義務**
- 服務完成後提供收費明細（紙本或電子）
- 每次服務前說明項目及費用，未同意費用不得收取
- 提供安全舒適空間，艙內空間足供寵物伸展
- 避免寵物遭受騷擾、虐待或傷害
- 運輸時注意食水、排泄、環境安全
- 未經甲方同意不得轉交他人照護
- **每日定時回傳寵物照片及照護情形**

---

## 四、意外/疾病與責任歸屬

- 發現異常應立即處理並通知甲方或緊急聯絡人
- 緊急就醫優先送往甲方指定獸醫院
- 未指定或未營業時，送往就近適當獸醫院
- **預設送醫：北投動物醫院；夜間急診：布達羊急診動物醫院**
- 可歸責於乙方之死亡或健康異常，乙方負損害賠償責任
- 代送就醫將依公開價目表收取陪診費、交通費

---

## 五、取消、退訂與逾時接回

### 住宿服務
- 訂金：NT$500
- 預定入住日前 **3 日以前** 取消：全額退還訂金
- 預定入住日前 **3 日內** 取消：訂金不退
- 入住後終止：扣除已用費用及訂金後退還餘款
- 逾時接回：依公開價目表住宿/安親價格計算逾時費及違約費

### 美容服務
- 服務前解除：不收手續費
- 服務後終止：依原價單價計算已使用費用後退還餘款
- 未經許可變更服務地點：甲方得解除/終止契約
- 完成後 **1 小時內** 須接回；逾時依公開價目表收安親費
- 超過營業時間：逾時費 **NT$300/小時**
- 超時嚴重者：乙方得通知動物保護主管機關
- 惡意隱瞞攻擊行為：乙方可終止契約；造成人員受傷得請求損害賠償
- 不可抗力（天災、政府法令）：乙方妥善安置，必要費用可向甲方請求償還
- 寵物因病、健康不佳、死亡或抗拒服務：雙方可解除/終止，乙方不扣手續費

- 甲方事後不同意，可通知乙方停止分享並刪除
